## Simulador simples de fisica 
### Inspiração: https://www.youtube.com/watch?v=_YbGWoUaZg0
Projeto para aprimorar minhas habilidades de POO em c++, alem de aprendizados de OpenGL e conceitos fisicos. 

Para compilar: 
```
g++ main.cpp interface.h corpoRigido.h engineFisica.h vector2.h -lglfw -lGL -o simulacao
```
